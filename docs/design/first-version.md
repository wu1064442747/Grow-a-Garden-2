# Grow a Garden 2 Guide First Version

Domain: `growagarden-2.online`

Positioning: a practical Grow a Garden 2 guide and tools site for launch-window
search demand.

First version scope:

- Homepage with fast entry points for codes, beginner guide, stock tracker, and
  high-demand guide pages.
- Independent SEO pages for codes, beginner route, seeds tier list, weather
  events, stealing defense, guilds, pets, and stock tracker.
- Static metadata, sitemap, robots, canonical URLs, and JSON-LD.
- Visual direction based on a clean garden guide interface: white background,
  green and blue accents, readable cards, compact navigation, and a simple
  code-native garden illustration.
